## UX Updates

### 1. Open Anyway — Shows Missing Count
**Before:** "Open anyway"
**After:** "Open anyway (3 songs missing)"
Users know exactly what they're accepting.

### 2. Re-Sync Feedback
**Before:** Same sync toast as initial sync
**After:** "3 songs updated — playlist improved" (shows improvement delta)
Fires 2s after sync completes so it doesn't overlap.

### 3. Skip Text
**Before:** "Skipped"
**After:** "Skipped — not in Spotify"
Explains why, reduces anxiety about missing songs.

### 4. Transition Feedback
**Before:** Spotify opened immediately (jarring)
**After:** "Opening Spotify…" toast → 250-300ms delay → opens
Both "Open anyway" and all-matched paths now have this brief transition.

## Edge Cases

- **0 failed on re-sync after fixing:** Improvement delta = original failures, toast shows full count
- **All songs skipped then re-sync:** Same failures, no improvement toast (delta = 0)
- **Single song missing:** "Open anyway (1 song missing)" — correct singular
