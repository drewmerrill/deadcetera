## Language Refinement

### Health Headlines (human, not robotic)

| Old | New |
|-----|-----|
| "Band is trending up and nearing gig-ready" | "The band is getting tighter" |
| "Momentum building — keep rehearsing" | "Momentum is building" |
| "Holding strong — band is in good shape" | "Band is in a good place" |
| "Steady but not growing — push harder" | "Holding steady — room to grow" |
| "Needs attention — recent sessions slipping" | "Recent sessions are slipping" |

### Coach Lines (new — italic, encouraging)

| State | Coach Line |
|-------|-----------|
| Improving + ready | "You're close — a couple more strong rehearsals and you're there." |
| Improving | "The work is paying off. Keep showing up." |
| Steady + strong | "Solid foundation — push for the next level." |
| Steady | "Consistency is good. Challenge yourselves this week." |
| Declining | "Not a crisis — just needs one focused rehearsal to turn it around." |
| No sessions | "Start tracking rehearsals to see the full picture." |
| Getting started | "Every rehearsal builds the foundation. Keep going." |

### Strengths (warm, specific)

| Old | New |
|-----|-----|
| "Consistent quality rehearsals" | "Quality is consistently high" |
| "Good time management" | "Staying on pace during rehearsals" |
| "Active rehearsal schedule" | "Rehearsing regularly — great rhythm" (3+/wk) or "Staying active with rehearsals" (2/wk) |
| "80% of songs gig-ready" | "80% of the setbook is gig-ready" |

### Issues (clear but not harsh)

| Old | New |
|-----|-----|
| "Sessions running long" | "Rehearsals running long — tighten transitions" |
| "No rehearsals this week" | "Haven't rehearsed this week — schedule one" |
| "N songs below readiness threshold" | "N songs need more reps" |
| (new) | "Recent sessions dipping — refocus next rehearsal" |

## Signal Priority

Strengths ordered: frequency → quality → time management → readiness %
Issues ordered: frequency gap → timing → rating dip → low readiness songs

Most actionable signals surface first.
