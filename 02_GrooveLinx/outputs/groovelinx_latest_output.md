# GrooveLinx — Band Feed: Links, Photos, Notes + Pin to Band Room

_Generated by Claude · 2026-03-24_

## Model

**Band Feed** = active sharing (all post types, real-time)
**Band Room** = durable memory (pinned items from feed)

## New Post Types

| Type | Create | Display |
|------|--------|---------|
| **Note** | Quick text | Text body |
| **Link** | Title + URL | Text + clickable link with domain label (YouTube, Spotify, etc.) |
| **Photo** | Caption + image URL | Text + inline image (max 240px height) |
| **Idea** | Title + optional link | Text + link (existing) |
| **Poll** | Question + options | Interactive voting (existing) |

Each post saves `post_type` field: 'note', 'link', 'photo' (or null for legacy).

## Pin to Band Room

Every post has a 📌 Pin / Unpin button in the action bar.
- Pin saves `pinned: true` to feed_meta
- Pinned posts show "📌 Pinned to Band Room" badge
- "📌 Pinned" filter tab shows all pinned items
- Band Room should read pinned items from feed_meta

## Edit Posts

Creator (or admin) can edit any post:
- ✏️ button opens inline edit area below the post
- Textarea pre-filled with current text
- Link field shown for link-type posts
- Save updates Firebase directly + sets `edited_at` timestamp
- Cancel removes edit area

## Filters

New filter tabs in filter bar:
- 🔗 Links — posts with links or post_type='link'
- 📷 Photos — posts with photo_url or post_type='photo'
- 📌 Pinned — items pinned to Band Room
- ⚙️ System — auto-generated posts (existing)

## Link Display

Links show smart domain labels:
- youtube.com → "YouTube"
- spotify.com → "Spotify"
- archive.org → "Archive"
- drive.google.com → "Google Drive"
- instagram.com → "Instagram"
- Other → shortened hostname

## Action Bar (per card)

```
[Resolve] [Tag] [Note] [Archive] [📌 Pin] [✏️ Edit] [🗑️ Delete]
```

- Pin/Edit/Delete: visible based on permissions
- Pin: toggles pinned state, shows active when pinned
- Edit: inline edit area with save/cancel
- Delete: confirmation + permanent removal (existing)

## Schema

New fields on feed items (`ideas/posts/{id}`):
- `post_type`: 'note' | 'link' | 'photo' (or null for legacy)
- `photo_url`: image URL for photo posts
- `edited_at`: ISO timestamp when edited

Feed metadata (`feed_meta/{type}:{id}`):
- `pinned`: boolean — pinned to Band Room
