## Chart System (Part 2)

### Firebase Schema
```
bands/{slug}/songs/{title}/chart_master   — original imported chart (string)
bands/{slug}/songs/{title}/chart_band     — band-edited version (string)
```

### Chart Panel (`renderChartPanel`)
- Song title + key + BPM header
- Monospace chord chart with syntax highlighting
- Section labels `[Intro]`, `[Verse]`, `[Chorus]` → blue bold
- Chord names (A, Bm, F#7, etc.) → yellow bold
- Edit button → inline textarea
- Import button → paste from Ultimate Guitar / Chordify
- Band chart shown by default; "View original" link if both exist

### Edit Flow
1. Tap ✏ Edit → textarea appears with current chart text
2. Edit inline (add sections, notes, chords)
3. Save → writes to `chart_band` in Firebase
4. Display updates immediately

### Import Flow
1. Tap 📥 Import → paste area appears
2. Paste from Ultimate Guitar or any text source
3. "Import as Master" → saves to `chart_master`
4. Panel reloads showing imported chart

### Setlist Chart Mode (`renderSetlistCharts`)
- Renders all songs in setlist order as continuous scroll
- Set headers as section dividers
- Each song: title + key/BPM + formatted chart
- "No chart available" for songs without charts
- `page-break-inside: avoid` for print friendliness

### Chord Highlighting Regex
Matches: A, Am, Bm7, F#, Cmaj7, Dsus4, G/B, etc.
```
/\b([A-G][#b]?(?:m|maj|min|dim|aug|sus|add|7|9|11|13|6)?(?:\/[A-G][#b]?)?)\b/g
```
