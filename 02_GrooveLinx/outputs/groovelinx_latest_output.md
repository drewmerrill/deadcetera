## Headline Insights

Each session card shows a bold, contextual headline:

| Condition | Headline |
|-----------|----------|
| Rating=great + only great this month | "Best rehearsal this month" |
| Rating=great | "Great session" |
| On-time + 4+ songs | "Tight and efficient session" |
| 10+ min over budget | "Ran long — transitions need work" |
| 60%+ blocks over | "Most songs ran over — tighten up" |
| 10+ min under | "Finished early — add more to the plan" |
| Rating=solid | "Solid session" |
| Rating=needs_work | "Needs work — keep pushing" |
| 60+ min + 6+ songs | "Deep work session — N songs in N min" |

Logic in `_rhGetHeadline(session, idx, allSessions)`. Replaces summary line when present.

## Trend Indicator

Top of Past Rehearsals section shows:

```
🔥 💪 💪 🔥 🔧    → Improving
```

- Last 5 session ratings as emoji dots (oldest → newest)
- Simple trend: **Improving** / **Steady** / **Needs attention**
- Calculated by comparing average of recent half vs older half
- Only shown when 2+ sessions exist

## Mixdown Tagging

Sessions with mixdowns show a cycling tag button:

| Tag | Display | Color |
|-----|---------|-------|
| None | "🏷 Tag" | Gray |
| best_take | "⭐ Best Take" | Gold |
| needs_work | "🔧 Needs Work" | Red |

Click cycles through: none → Best Take → Needs Work → none.
Saved to `session.mixdown_tag` in Firebase.
