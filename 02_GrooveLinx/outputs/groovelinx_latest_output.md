# GrooveLinx — Rehearsal Session Lifecycle + Mixdown Integration

_Generated by Claude · 2026-03-24_

## Changes

### 1. Session Summary Screen

After "End Rehearsal", a **full-screen summary overlay** appears:

- Total duration (bold, centered)
- Songs worked count
- Performance vs plan (+/- minutes, color-coded)
- Song list
- **Notes textarea** — "How did it go? What needs work?"
- **Mixdown attachment** — paste Drive URL or upload MP3
- **Save Session** button (finalizes notes + mixdown)
- **Skip** button (session already saved, just closes)

The session is saved to Firebase on rehearsal close. The summary screen lets the user enrich it with notes and mixdown before moving on.

### 2. Mixdown Attachment Flow

From the summary screen:
- Paste a Google Drive link → stored as `drive_url`
- Upload MP3 → creates blob URL for playback
- Both create a mixdown record in `rehearsal_mixdowns/{id}` with `linked_session_id`
- Session updated with `mixdown_id`

From Past Rehearsals:
- Sessions with mixdowns show 🎤 icon
- "🎤 Play Mixdown" button opens Rehearsal Chopper with the audio loaded

### 3. Unified Past Rehearsals

**Removed**: separate "Last Rehearsal" review section (`_rhRenderSessionReview`)

**Unified**: single "Past Rehearsals" list with:
- Most recent session highlighted (left border + "LATEST" badge + tinted background)
- All sessions in one clean list (no duplicate "Last Rehearsal" card)
- Each card: date, duration, songs, verdict, notes, mixdown indicator
- Details expand, Delete, Play Mixdown actions

### 4. Session Lifecycle

```
Plan → Start Rehearsal → Active Session → End Rehearsal → Summary Screen → Past Rehearsals
```

| Phase | What Happens |
|-------|-------------|
| Plan | User builds rehearsal plan with songs + time budgets |
| Start | `_rhLaunchSavedPlan()` → opens rehearsal mode |
| Active | Timer tracks per-block timing, songs worked |
| End | `closeRehearsalMode()` → saves session → shows summary |
| Summary | User adds notes + attaches mixdown → finalizes |
| History | One clean card per session in Past Rehearsals |
