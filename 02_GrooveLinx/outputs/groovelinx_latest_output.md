## 3. Button Feedback

`addButtonFeedback(btnId)` — adds subtle press animation:
- `mousedown`/`touchstart`: scale(0.95) + opacity 0.8
- `mouseup`/`touchend`/`mouseleave`: resets

Available for any playback button via `PlaybackSession.addButtonFeedback('btnId')`.

## 4. Analytics Logging

`_logPlayback(event, source, durationMs)`:
- Logs to console: `[Playback] attempt YouTube 0ms`
- Sends to FeedMetrics if available: `playback_attempt`, `playback_success`, `playback_error`, `playback_timeout`
- Tracks time-to-playback (ms from STARTING to terminal state)

Events: `attempt` → `success` | `error` | `timeout`

## 5. Preload Structure

`_preloadHint(sorted, nextIdx)`:
- On successful play, DNS-prefetches the next track's domain
- Adds `<link rel="dns-prefetch" href="//open.spotify.com">` to head
- No-op if next track unavailable or same domain already prefetched
- Future: YouTube embed preload, Spotify widget warmup
