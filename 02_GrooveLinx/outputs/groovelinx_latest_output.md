## UI Decisions

- Replaced old 3-step progressive onboarding (steps 1/2/3 with different messages per visit) with single clear explainer that shows 3 times then disappears. Simpler, more informative per impression.
- Feed explainer uses structured layout (icon + bold label + arrow + description) instead of paragraph text. Skimmable in < 3 seconds.
- Mode guidance is italicized, dimmed, small â€” doesn't compete with dashboard content.
- First-action pulse is CSS-only (no JS animation loop), runs 3 times then stops.
- No modals. No forced walkthroughs. No multi-step flows.
