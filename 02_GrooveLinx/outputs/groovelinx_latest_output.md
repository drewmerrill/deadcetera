
## Player Flow

```
User taps headphones
  → GLPlayerEngine.loadFromSetlist(setlist, options)
  → GLPlayerUI.showOverlay()
  → GLPlayerEngine.play(0)

play(idx):
  → emit songChange
  → check caches (instant path)
  → if cached → embedReady → PLAYING
  → else → RESOLVING

RESOLVING:
  → GLSourceResolver.resolve(title, band, { mode, onStatus })
  → onStatus: "Trying YouTube..." → "Trying Spotify..." → "Trying Archive..."
  → chain: preferred → Spotify → YouTube → Archive
  → silent retry on first source
  → 4s timeout
  → result? → embedReady → PLAYING
  → no result? → FALLBACK

FALLBACK:
  → Retry button
  → Paste YouTube URL
  → Skip button
```

## Player States

| State | When | UI |
|-------|------|-----|
| IDLE | No queue loaded | Nothing visible |
| LOADING | Queue being built | "Loading..." |
| RESOLVING | Finding source for current song | "Trying YouTube..." etc. |
| PLAYING | Source embedded and active | Video/audio playing |
| FALLBACK | All sources failed | Retry / paste / skip UI |
| ERROR | System error | Error message + retry |

Transitions are deterministic. Every state has an exit path. No dead ends.

## Fallback Chain

### Default Mode
1. Preferred source (configurable)
2. Spotify
3. YouTube
4. Archive

### Jam Mode
1. Archive (live recordings)
2. Spotify
3. YouTube

## Resolution Priority (per song)

1. Set-level override (passed in options)
2. Curation: override_id
3. Curation: north_star_id
4. Curation: selected_id
5. Firebase saved versions (spotify_versions)
6. Local cache (localStorage)
7. Auto-search (YouTube: 7 parallel backends, Spotify: search API, Archive: advanced search)

## UI Modes

### Overlay (Full-Screen)
- Setlist header + source preference dropdown
- Video/audio container (16:9)
- Song title + artist + source label + confidence badge
- Car-friendly controls (80px play button)
- Up next bar
- Minimize → Bar

### Float (Mini-Player)
- Floating card over charts/content (280px wide)
- Compact video + title + controls
- Expand → Overlay
- Chart remains scrollable beneath

### Bar (Now-Playing)
- Fixed bottom bar
- Play/pause + title + skip + close
- Tap → Overlay
- Safe-area aware

## Confidence Labels

| Label | When |
|-------|------|
| Best match | Curated, cached, or Firebase-saved version |
| Close match | Auto-search result |
| Live version | Archive.org result |

## Reliability Guarantee

Every playback attempt ends in:
- In-app playback (YouTube/Spotify/Archive embed)
- OR in-app fallback UI (retry / paste URL / skip)
- NEVER indefinite loading
- NEVER silent failure
- NEVER stuck state
