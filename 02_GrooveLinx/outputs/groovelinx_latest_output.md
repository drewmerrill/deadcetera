# GrooveLinx — Rehearsal Mixdowns

_Generated by Claude · 2026-03-24_

## Feature

Session-level rehearsal recording archive. Upload MP3 or paste Drive link. In-app audio player. One-click Rehearsal Chopper integration.

## Schema

Firebase: `bands/{slug}/rehearsal_mixdowns/{id}`

  "created_at": "2026-03-20T22:00:00Z",
  "updated_at": "2026-03-20T22:00:00Z",

## UI

Renders on the Rehearsal page below session history. Each mixdown card shows:

- Title + date
- Duration + linked setlist (if any)
- Notes preview
- In-app `<audio>` player (play/pause/scrub)
- Action buttons: Chopper, Open in Drive, Copy Link, Edit, Delete

"+ Add Mixdown" button opens inline form with:
- Title + date fields
- Notes
- Google Drive share link input
- Direct audio URL input
- File upload (MP3/WAV/M4A/etc.)
- Duration + linked setlist name
- Save / Cancel

## Behavior

### Add Mixdown
1. Paste Drive link and/or direct audio URL
2. Or upload MP3 file (creates local blob URL for session playback)
3. Save → persists to Firebase

### Playback
- Direct audio URL → in-app `<audio>` player (native HTML5)
- Drive link only → "Open in Drive" button
- Both → audio player + Drive link

### Chopper Integration
Every card with audio shows "✂️ Chopper" button:
1. Opens Rehearsal Chopper modal (`openRehearsalChopper()`)
2. After 300ms (modal render), loads audio URL into `#chopAudio`
3. Shows toast "Loaded: {title}"

### Copy Link
Copies audio_url or drive_url to clipboard.

### Delete
Confirmation dialog → removes from Firebase.
