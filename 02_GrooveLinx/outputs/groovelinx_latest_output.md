## Completion Reflection

Each completion screen now shows a context-aware reflection line:

| Context | Reflection |
|---------|-----------|
| Weak songs, 5+ songs | "Deep focus session — your weak spots are getting stronger" |
| Weak songs, <5 | "Targeted practice — every rep counts" |
| Gig set, 10+ songs | "Full run-through — you're gig-ready" |
| Gig set, <10 | "Tight run — ready for the stage" |
| 8+ songs | "Strong session — stayed locked in" |
| 4+ songs | "Good work — building momentum" |
| Default | "Every practice counts — keep showing up" |

Displayed in italic below the headline, before the song count.

## Streak on Completion

If streak ≥ 2, shown as a pill:

```
🔥 5 days in a row          (≥5, red)
⚡ 3 days in a row          (3-4, amber)
💪 2 days in a row          (2, indigo)
```

Footer also adapts: "🔥 5-day streak — don't break it" (if ≥3).

## Band Signal on Completion

Light band-level message derived from rehearsal ratings:

- "↑ Band trending up this week" (improving)
- "→ Band holding steady" (stable)

Shown below streak, above next actions.
