# GrooveLinx — Live Gig Mode Refinements

_Generated by Claude · 2026-03-25_

## Changes

### 1. Audio Button Hint
First-time users see a tooltip below the 🎧 button: **"🎧 Play audio with charts"**. Appears 800ms after entering Go Live. Dismisses after 4 seconds. Shows once (persisted via `gl_lg_audio_hint_seen`).

### 2. Mode Transition
Overlay fades in with 250ms opacity transition on entry. No abrupt appearance.

### 3. Current Song Emphasis
- Title bumped from 1.1rem to **1.3rem** with subtle indigo text-shadow
- **Pulse animation** on song change: background flashes indigo for 400ms then fades
- "+ Note" button added to song bar (right-aligned, subtle)

### 4. Up Next Visibility
- Label upgraded: **"COMING UP →"** in indigo (was "UP NEXT:" in gray)
- Next song shown in **bold white** with key badge if available
- Second-next song still visible but dimmer
- Last song: **"🎶 LAST SONG — FINISH STRONG"** in green
- Top border in indigo (or green for last song) for visual separation

### 5. Quick Notes
"+ Note" button in the song bar toggles an inline input:
- Text input pre-labeled with current song name
- Enter or Save button to commit
- Saved to `_lgNotes[]` array in session
- Persisted to Firebase: `live_gig_notes/{setlistId}`
- Toast confirmation: "Note saved for Scarlet Begonias"

Notes are per-session, tied to the setlist. Can be reviewed post-gig.
