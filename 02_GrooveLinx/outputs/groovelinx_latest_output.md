## Fix 5 — Timeout Safety Net
5-second timer starts in STARTING state. If still STARTING after 5s:
- State → ERROR
- Toast: "⚠️ Taking too long — try YouTube or Spotify"
- Fallback overlay with manual search links

Timer cleared on success or catch.
