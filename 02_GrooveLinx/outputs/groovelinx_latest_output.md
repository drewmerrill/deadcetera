## Recommendation

### DO: Extend Play Dashboard + Wire Float Player

```
1. Add _renderNextActionCard + _renderBandScorecard to _renderPlayDashboard
   (same as Sharpen — 2 function calls)

2. In Live Gig Mode "Go Live", auto-launch GLPlayerUI.showFloat()
   so audio plays as mini-player while charts scroll beneath
   (~10 lines in live-gig.js)

3. Add a "Play Audio" toggle button to Live Gig Mode header
   that starts/stops the float player

4. SetlistPlayer: add guard to not render if GLPlayerUI is active
   (prevent z-index conflict)
```

### DO NOT:
- Create a new "Band Mode" overlay
- Duplicate the player engine
- Build a parallel navigation system
- Create new Firebase schemas for this

### Result:
"Band Mode" = **Play mode dashboard (guided) + Go Live with floating audio player**

No new pages. No new overlays. Just wiring.
