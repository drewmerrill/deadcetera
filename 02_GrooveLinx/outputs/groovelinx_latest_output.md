## Completion Screen

When the queue finishes, the player overlay transforms into a completion screen:

```

You practiced 8 songs
Practicing weakest songs (8)

    What's next?
[🏠 Back to Home    ]
[🔁 Run It Again    ]
[🎸 Start Rehearsal ]
```

- Animated checkmark (bounce)
