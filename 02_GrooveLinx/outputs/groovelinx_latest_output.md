## UI Changes

### 1. Creation Bar — Messaging Style
**Before:** Input + dashed "Poll, Idea, or more" button
**After:** Card-style container with:
- Rounded pill input ("Share something with the band…")
- Circular send button
- Three equal type buttons below: 📝 Note | 💡 Idea | 🗳️ Poll
- Feels like a messaging app, not a form builder

### 2. Ownership Labels — Explicit
**Before:** Subtle badge text ("Your input needed")
**After:** Added explicit ownership line below card body:
- "⚡ You owe this" (yellow, bold) — when `needsMyInput === true`
- "Needs input from: Brian, Chris" (indigo) — for targeted items
- "Waiting on: Jay, Pierce" (dim) — for polls you voted on

These are **in addition to** the header badge, not replacing it. The badge classifies; the label directs.

### 3. Targeting Label
**Before:** "For: Chris, Brian"
**After:** "Needs input from: Chris, Brian" — action-oriented, not passive

### 4. Action Confirmations
**Before:** "Vote recorded", "Acknowledged"
**After:** "✅ Vote recorded", "✅ Acknowledged" — checkmark prefix for visual completion signal

### 5. Targeting Header in Forms
**Before:** Small dim "Who needs to respond?"
**After:** Bolder, more prominent — `font-weight:700`, `color:var(--text-muted)` instead of `var(--text-dim)`
