## PART 1 — FIRST REHEARSAL EXPERIENCE

### Flow (8 steps, routes into existing systems)

| Step | Surface | What Happens |
|------|---------|-------------|
| 1. Welcome | Home (first visit) | Avatar tip: "Welcome to GrooveLinx! Let's start by adding a few songs." |
| 2. Minimum Setup | Songs page | Starter pack import OR manual add. Avatar tip triggers when < 3 songs. |
| 3. Readiness | Songs page | Existing readiness rating on song detail. Avatar doesn't interrupt — waits. |
| 4. First Practice | Home → Practice Set | Avatar tip: "Nice — you've got songs in. Want to run a quick practice set?" → launches `hdPlayBundle('focus')` into existing unified player |
| 5. Feedback | Completion screen | Existing completion flow: rating (Great/Solid/Needs Work) + reflection + streak |
| 6. Rehearsal Intro | Home after practice | Avatar tip: "You just ran your first practice set! Next step: try a full rehearsal." |
| 7. First Completion | Rehearsal summary | Existing session summary: duration, songs, rating, notes, mixdown |
| 8. Home After | Home dashboard | Next Action Card + Scorecard + Top Songs visible. Avatar quiet. |

**No new pages. No new overlays. Just Avatar tips routing into existing flows.**

## PART 3 — RULE-BASED TRIGGERS (15 BUILT)

| ID | Stage | Trigger | Message |
|----|-------|---------|---------|
| welcome | fan | first_visit | "Welcome to GrooveLinx! I'm here to help your band get tighter." |
| empty_songs | fan | no_songs (< 3) | "Your setbook is empty — add 3–5 songs to get started." |
| first_songs_added | fan | songs ≥ 3 | "Nice — you've got songs in. Want to run a quick practice set?" |
| empty_setlist | fan | no setlists | "No setlists yet. Build one for your next gig or rehearsal." |
| first_practice_done | fan | first completion | "You just ran your first practice set! That's how bands get tighter." |
| practice_nudge | fan | not practiced today | "Haven't practiced today. Even a quick run helps." |
| weak_songs_exist | bandmate | weak > 2 | "{weakCount} songs still need work before the gig." |
| gig_soon | bandmate | gig ≤ 2 days | "Gig in {daysOut} day{s}. Run the set once more." |
| post_rehearsal | bandmate | just finished | "Good session. Don't forget to add notes or attach the mixdown." |
| no_rehearsal_this_week | bandmate | 0 this week | "No rehearsal logged this week. Schedule one?" |
| scorecard_issue | bandmate | has issues | "Scorecard flagged: {topIssue}" |
| transitions_slow | coach | sessions long | "Recent rehearsals are running long. Transitions might need tightening." |
| band_improving | coach | trend improving | "The band is getting tighter. Keep this rhythm going." |
| keep_going | coach | practiced + weak | "Good work today. {weakCount} songs still need attention." |
| mixdown_reminder | bandmate | no mixdown | "Last rehearsal has no recording attached. Got a mixdown to upload?" |

### Cooldowns

| Tip | Cooldown |
|-----|----------|
| welcome, first_songs, first_practice | Once ever |
| practice_nudge, weak_songs, keep_going | 12 hours |
| gig_soon | 6 hours |
| empty_songs, empty_setlist, no_rehearsal, scorecard, mixdown | 24 hours |
| transitions_slow, band_improving | 48 hours |

## PART 4 — GUIDANCE LIBRARY STRUCTURE

```javascript
{
  id: 'weak_songs_exist',
  stage: 'bandmate',           // fan | bandmate | coach
  trigger: 'has_weak_songs',   // evaluated by _checkTrigger()
  page: 'home',                // page context filter
  message: '{weakCount} songs still need work before the gig.',
  coach: 'Focus on the weakest ones first.',  // optional italic line
  actions: [{ label: 'Practice Weak Songs', onclick: "hdPlayBundle('focus')" }],
  cooldown: 43200000,          // ms before re-showing
  dismissible: true
}
```
