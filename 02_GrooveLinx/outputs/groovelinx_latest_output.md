## Where Help Recall Is Added

**Location:** Feed page header, right-aligned circle button with "?"
**Behavior:** Toggles the explainer banner on/off. Does not reset visit counter. Tap once to show, tap again (or X) to hide.

## How Micro-Guidance Works

1. First item in "YOU OWE INPUT" gets:
   - Amber pulse CSS animation (3 cycles)
   - "→ Tap to complete this" label above the card content
2. User taps any action (vote, acknowledge, resolve, navigate):
   - `_feedAdvanceOnboarding()` fires
   - Pulse class removed from card
   - Micro-guide label removed from DOM
   - `localStorage('gl_first_action_highlight_seen')` set
   - Momentum toast shown: "Nice — 2 left" (or "Nice — keep going")
3. Next feed visit: no highlight, no micro-guide (one-time only)

## Edge Cases

- **No actionable items**: No highlight or micro-guide rendered (only shows when `needsMyInput > 0` and items exist in "I Owe" group)
- **Help recall when explainer already visible**: Toggles it off (removes it)
- **Momentum nudge count**: Uses `computeSummary().needsMyInput - 1` since the action was just completed but feed hasn't re-rendered yet
- **Zero remaining after first action**: Shows "Nice — keep going" instead of "Nice — 0 left"
