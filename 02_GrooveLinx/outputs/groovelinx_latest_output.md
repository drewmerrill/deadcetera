# GrooveLinx — Guided Band Operating System

_Generated by Claude · 2026-03-24_

## Goal
Every screen answers: "What should I do next?"

## 1. Next Action Card (Home Decision Engine)

Bold card at the top of every dashboard. Decision tree:

| Condition | Title | CTA |
|-----------|-------|-----|
| Gig today | "Showtime" | Go Live |
| Gig in 1-2 days | "Gig in N days" | Practice Set |
| Workflow state has next action | Dynamic from GLStore | Go |
| Weak songs exist | "N songs need work" | Start Practicing |
| No weak songs | "You're in good shape" | Practice Set |

Renders with icon, title, subtitle, and gradient CTA button.

## 2. Top Songs to Work

New card below Next Action:
- Shows top 3 songs by lowest readiness score
- Color-coded urgency (red ≤2, amber ≤3, gray >3)
- "Practice All" button launches focus bundle via unified engine
- Only shows for songs in active status (prospect, learning, rotation, gig_ready)

## 3. Practice Mode Clarity

Listen & Learn reframed:
- **Sharpen**: "Practice Your Set — Listen to your weakest songs and lock them in"
- **Lock In**: "Rehearsal Prep — Listen to what we're working on"
- **Play**: "Play Your Set — Listen through the gig setlist"

Clear purpose. No ambiguity.

## 4. Player Confidence Messaging

### Status messages (during resolution)
- "Loading..." (immediate, on tap)
- "Trying YouTube..." / "Trying Spotify..." / "Trying Archive..."
- "Searching for a version..." (fallback state)

### Source labels (when playing)
- "▶ Playing now · YouTube · Best available version"
- "🎵 Playing now · Spotify · Found version"
- "🏛️ Playing now · Archive · Live recording"

**"Playing now"** not "Playing via". Confidence, not uncertainty.

### Confidence labels updated
| Old | New |
|-----|-----|
| Best match | Best available version |
| Close match | Found version |
| Live version | Live recording |

