## Summary

| File | What |

## Spotify Button States

| State | Label | Style |
|-------|-------|-------|
| Not connected | "🎵 Connect Spotify" | Green border + bg |
| Connected, no playlist | "🎵 Create Playlist" | Green border + bg |
| Connected, playlist exists | "🎵 Sync to Spotify" | Dim green, no bg |

Checks `isSpotifyConnected()` + `gl_spotify_playlists[bundleType]` at render time.

## Play Confirmation Overlay

After quick launch resolves matches:

```
┌──────────────────────────────┐
│    6 of 10 songs ready       │
│    Starting with: Althea     │
│                              │
│    [▶ Play first track]      │
│    [📺 YouTube] [🎧 Spotify] │
│                              │
│          Close               │
└──────────────────────────────┘
```

- "Play first track" opens URL in new tab (user-gesture context — no popup block)
- YouTube/Spotify links open search for first song
- No silent auto-open

## Next Song Bar

After playing first track, if more songs in bundle:

```
Up next: Sugaree · 2/10    [Next Song →] [✕]
```

- Fixed bottom bar (z-index 9400)
- "Next Song →" opens next resolved URL
- Tracks position through sorted URLs
- Last song: "Last song" + close
- "All songs played" toast at end
