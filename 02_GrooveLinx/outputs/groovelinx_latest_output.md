
### When No Embed Available
Standard text bar: "Up next: Sugaree · 2/10 [Next →] [✕]"

## Embed Technical Details

Official Spotify oEmbed iframe:
```html
<iframe src="https://open.spotify.com/embed/track/{id}?utm_source=generator&theme=0"
  width="100%" height="152" frameBorder="0"
  allow="autoplay; clipboard-write; encrypted-media; fullscreen; picture-in-picture"
  loading="lazy" style="border-radius:8px">
</iframe>
```

- **Full embed:** 152px height (play overlay)
- **Compact embed:** 80px height (next-song bar)
- **Theme:** Dark (theme=0)
- **No auth required** — works for any public track
- **Limitations:** User must tap play; no programmatic autoplay

## Limitations Copy
