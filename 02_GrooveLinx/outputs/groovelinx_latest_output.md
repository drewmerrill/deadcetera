## Momentum Reinforcement

**Trigger:** 3rd action completed in a single session
**Frequency:** Once per session (sessionStorage)
**Does not stack:** Fires alongside the regular momentum nudge ("N left — keep going") but the momentum reinforcement only fires once, on the 3rd action specifically

## Edge Cases

- **Session < 3 actions:** Momentum never fires
- **Momentum + existing nudge:** The 3rd action will show the momentum toast; the regular nudge ("N left") fires from `_feedAdvanceOnboarding` as usual. Since momentum fires first (sync, before the rerender nudge), they appear sequentially without stacking.
