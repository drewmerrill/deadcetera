## Phase 1 — Attention Summary Bar

Top of the feed page, above filter buttons. Two clickable cards:

- **Red dot + count** — Critical items. Clicking filters to `critical`.
- **Yellow dot + count** — Need Your Input (unresolved `needs_input`). Clicking filters to `needs_input`.

If both counts are zero, shows a green "All clear" message instead.

**How counts are calculated:**
- Excludes archived items
- Critical = items where effective tag (meta override or original) is `mission_critical`
- Need Your Input = items where effective tag is `needs_input` AND not resolved (meta override or original)
- Counts update live when tags/resolve/archive actions are taken (all three bars re-render on every action)

## Phase 2 — Return Navigation

When a user clicks through a feed item to a deep-linked page (Songs, Band Room), a **fixed top bar** appears:

```
[← Back to Feed]
```

- Fixed to top of viewport with blur backdrop
- z-index 900 (below modals, above content)
- Single tap returns to feed page
- Auto-removed when:
  - User navigates back to feed
  - User navigates away via nav rail (not from feed click-through)
  - Feed page re-renders

**How it works:**
- `_feedNavigatedFrom` flag tracks if current deep-link came from feed
- `_feedShowBackBar()` injects a fixed `#feedBackBar` element into `document.body`
- GLStore `pageChanged` subscription removes bar if navigating to feed or if navigation wasn't from feed
- `_feedBackToFeed()` calls `showPage('feed')` for 1-tap return

## Phase 3 — Note Visibility

Notes on feed items now have a layered display:

1. **Most recent note** — shown prominently with larger text and stronger color
2. **New note indicator** — notes less than 24 hours old get a purple dot and highlighted text (`#a5b4fc` author, white text)
3. **Older notes** — collapsed behind a "View X older notes" toggle button
4. **Toggle** — clicking expands/collapses older notes in place. Button text updates to "Hide older notes" when expanded.

**Display rules:**
- 0 notes: no notes section shown
- 1 note: shown prominently, no toggle
- 2+ notes: latest shown, rest behind toggle in reverse chronological order (newest first)
