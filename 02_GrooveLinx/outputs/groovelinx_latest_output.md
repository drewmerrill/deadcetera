## Summary

Surfaced FeedActionState outside Band Feed onto the home dashboard. Added inline idea acknowledgment. Strengthened completion moment. All surfaces now share the same action count source of truth.

## Files Changed

| File | What |
|------|------|
| `js/features/home-dashboard.js` | "You Owe" action card on all 3 mode dashboards (Sharpen, Lock In, Play) |

## Where Action State Is Surfaced

| Surface | What Shows | Source |
|---------|-----------|--------|
| Band Feed attention bar | "You owe input (N)" / "You're locked in" | `FAS.computeSummary()` |
| Band Feed "I Owe" filter | Items where needsMyInput=true | `FAS.getActionState()` |
| Left rail Feed badge | Personal action count | `FAS.setActionCount()` |
| Left rail Band Room badge | Unvoted poll count | `FAS.getMyVoteKey()` |
| **Home Dashboard (NEW)** | "N items need your input" + top 3 items, or completion state | `FAS.computeSummary()` via lightweight poll query |
| App icon badge | Same count | `navigator.setAppBadge()` |

All counts come from the same engine. No parallel logic.

## Inline Acknowledgment

**Ideas** with `needsMyInput=true` now show a green "Got it" button directly on the feed card.

Behavior:
- Tap "Got it" → saves an "Acknowledged" note with author + timestamp → marks item resolved via feed_meta
- Item immediately moves from "I Owe" → resolved (dims, badge changes to "Complete")
- Nav badge decrements
- No page navigation required

## Completion State

When `needsMyInput === 0`:

**Feed attention bar**: Gradient background, circular checkmark icon, two-line layout:
- "You're locked in." (bold green)
- "Nothing needs you right now." (subdued)
- "Waiting on band: X items" (if applicable)

**Home dashboard card**: Same messaging with green left border:
- Checkmark + "You're locked in."
- "Waiting on band: N" if applicable

Both update instantly after actions complete.

## Edge Cases

- **Feed not yet visited**: Dashboard card does lightweight poll-only Firebase query (same as background badge refresh) instead of requiring full feed load
- **Offline**: Shows "Offline" in dashboard card rather than hanging
- **No FeedActionState**: Graceful fallback "Feed not available"
- **Idea acknowledgment is non-destructive**: saves to feed_meta overlay, original idea data untouched
