## Match Confidence

After sync, toast shows detailed feedback:

| Scenario | Message |
|----------|---------|
| All matched | "15 songs synced — opening Spotify" |
| Some failed | "12 songs synced, 3 need review — opening Spotify" |
| Already synced | "Already up to date — opening Spotify" |
| No matches | "No Spotify matches found for any song" |
| Empty bundle | "No songs found for Rehearsal Prep" |
| Token expired | "Spotify session expired — tap Sync again to reconnect" |
| API error | "Spotify sync failed — try again" |

`syncToSpotify()` returns detailed result:
```json
{
  "ok": true,
  "matched": 12,
  "locked": 3,
  "searched": 2,
  "failed": 3,
  "failedSongs": ["Dark Star", "St. Stephen", "Terrapin Station"],
  "playlistId": "3cEYpjA..."
}
```

## Version Locking

**Data structure** (stored in song's `spotify_versions` array):
```json
{
  "spotifyTrackId": "4iV5W9uYEdYUVa79Axb7Rh",
  "spotifyMatchLocked": true,
  "url": "https://open.spotify.com/track/...",
  "platform": "spotify"
}
```

**Resolution priority:**
1. Locked track ID (`spotifyMatchLocked: true`) — highest confidence, skip search
2. Extract ID from saved version URL
3. Search Spotify API as fallback
4. Mark as failed if no match

Locking UI is a future enhancement — the data structure is ready now.

## Token Improvements

**Before:** Token expired → "Spotify session expired — reconnect" → full re-auth
**After:**
1. `_ensureValidToken()` called before every sync
2. If token expires within 5 minutes, attempts silent refresh via `refresh_token`
3. If API returns 401, retries once after refresh
4. Only forces reconnect if refresh fails
5. Reduces forced re-authorizations from ~hourly to ~never (refresh tokens last months)

## "Already Up To Date"

Tracks hash of last synced track URIs per bundle type. If bundle hasn't changed since last sync:
- Skips API calls
- Shows "Already up to date — opening Spotify"
- Opens playlist directly

## Error Visibility

Every failure path now shows a visible toast:

| Failure | Message |
|---------|---------|
| Bundle computation error | "Could not load songs" |
| Track resolution error | "Error matching songs: [detail]" |
| Playlist creation failed | "Could not create Spotify playlist" |
| Track PUT failed | "Spotify sync failed — try again" |
| No user (token invalid) | "Spotify session expired — tap Sync again" |

No silent failures. Every error is visible and suggests next action.
